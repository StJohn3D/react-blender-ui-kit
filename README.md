#Timber UI

Setup

cd timber-ui/timber-ui

npm install

npm run build

npm link

then...

cd timber-ui/demos

npm install

npm link timber-ui

npm start

Now go look at localhost:3000

Also install the Redux dev tools in chrome if you haven't already
